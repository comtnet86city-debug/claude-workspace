# claude-workspace

`~/CLAUDE.md` 的 GitHub 備份——Claude Code CLI 工作區設定文件。

## 內容

`CLAUDE.md` 是 Claude Code 自動讀取的指引文件，記錄：

- **環境** — 虛擬環境、LLM 模型、外部服務、OpenClaw Gateway
- **腳本用法** — PPT / Excel / RAG / Whisper / ComfyUI / TTS / Vision 等
- **專案結構** — 所有目錄與關鍵檔案說明
- **Git Repositories** — 所有相關 repo 清單與 CLAUDE.md 涵蓋內容
- **程式架構** — CrewAI 模式、RAG 管線、各服務架構
